# pyPDFLibrarianSort - Project Brief

## 30-Second Overview

**AI-powered PDF library organizer** that automatically categorizes and renames PDFs using AI (Gemini, Anthropic, or DeepSeek). Features batch processing for 100x cost savings, automated watch mode, and a modern web interface.

## Key Innovation

**Batch Processing**: Process 200 PDFs in one API call ($0.10) vs individual processing ($10). **98% cost savings**.

## Core Capabilities

1. **Three AI Providers**: Gemini, Anthropic, DeepSeek
2. **Batch Mode**: Process all PDFs in single API call
3. **Watch Mode**: Auto-organize PDFs as they arrive (24/7)
4. **Web Interface**: Drag & drop, real-time categorization, visual approval
5. **Smart Renaming**: Detects gibberish filenames, reads content, suggests meaningful names

## Target Users

- Researchers with hundreds of PDFs
- Students organizing study materials
- Professionals managing document libraries
- Anyone drowning in unorganized PDFs

## Technical Stack

- **Language**: Python 3.x
- **AI Providers**: Google Gemini, Anthropic Claude, DeepSeek
- **PDF Processing**: pypdf
- **File Monitoring**: watchdog
- **Web Framework**: Flask
- **Deployment**: Cross-platform (Windows, macOS, Linux)
